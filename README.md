# UC14-HTML-ECSS
<h1>Meu Jogo Favorito: Among Us</h1>

<h2>Sobre o jogo</h2>
<p>Among Us é um jogo onde você joga com várias pessoas dentro de uma nave, e nem todo mundo é confiável. Alguns são tripulantes tentando completar tarefas, enquanto outros são impostores que precisam sabotar tudo e eliminar os outros sem serem descobertos.</p>

<h3>Como funciona</h3>
<p>Durante o jogo, você precisa prestar muita atenção no que os outros fazem. Quando alguém encontra um corpo ou chama reunião, todo mundo começa a discutir e tentar descobrir quem é o impostor. A parte mais legal é justamente ess ninguém sabe quem está mentindo.</p>

<h4>Por que eu gosto</h4>
<p>Eu gosto de Among Us porque não é só sobre jogar, é sobre enganar e descobrir mentiras. Quando jogo com meus amigos, fica muito mais engraçado, porque todo mundo começa a se acusar e tentar se defender. Cada partida é diferente, nunca fica repetitivo.</p>

<h5>Dica básica</h5>
<p>Se você for impostor, não fica matando toda hora, porque isso entrega rápido. O melhor é fingir que está fazendo tarefa e agir como se fosse inocente.</p>
